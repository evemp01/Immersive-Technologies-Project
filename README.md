# Immersive-Technologies-Project
## GAME IDEA

PLAYERS:
-a- Sees a map from above
-b- Lives in the 3D map


BASE LOGIC:
- Player -a- has to guide player -b- around the map.
- Player -b- does not have any idea about what's going on in the map, he just has to follow the instructions.
- Monsters are running around in the maze, if they touch player -b- the game ends.
- If Player -b- is able to collect all the keys is the maze, and get to the EXIT door, the game ends/moves to the next level.

MECHANICS:
- The points you make are relative to the time it took to exit the room;

- Different ghosts have different stats [moving speed/pathfinding towards the player];

- To use the keys, Player -b- needs to interact with the doors: choose the right key for the right lock;


INTERRACTIONS BETWEEN PLAYERS:
- The glass below Player -a- gets dirty with time: he has to clean it with the mop otherwise he cannot suggests Player -b-;

- Player -b- has to collect the "mop coins" around the mape and send them to Player -a- though a given gesture/button/elevator.
